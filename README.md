## Project setup

Este proyecto está hecho con [Expo](https://docs.expo.io/)

```markdown
### Instal
npm install

### Compiles and hot-reloads for web
npm run web

### Open Developer Tools on browser and choose iPhone X mobile, for example

### OR

### Compiles and hot-reloads for android emulator
npm run android
```
